About the Project

  • 프로젝트 기간:

    2025.08 ~ 2025.12

  • 프로젝트 인원:

    4명

  • 역할 (담당 업무):

    1. UI 퍼블리싱 전반
    2. 다국어 언어팩 시스템 구축
    3. 메인 페이지 및 전역 레이아웃(헤더·푸터·사이드바) 개발
    4. 급여 조회, 연말정산, 헬프데스크 개발

  • 프로젝트 설명:

    EY한영 내부 직원들을 위한 모바일 기반 통합 관리 시스템 구축 프로젝트로, 전자결재 신청, 근태, 급여·연말정산 조회, 헬프데스크 등 다양한 내부 업무 기능을 통합한 시스템 개발을 진행하였습니다.

  • 기술 스택:

    - Next.js
    - Typescript
    - TanStack Query (서버 상태 관리)
    - Zustand (클라이언트 상태 관리)
    - Tailwind CSS (UI 스타일링)
